## Aufgabe Es soll eine Ampelschaltung gebaut werden. Die Ampel wird mit Hilfe einer Website geschaltet. Der Workshop besteht dabei aus drei Teilaufgaben und dem Aufbau der Schaltung. ## Inventar * Raspberry Pi * Steckbrett * 3x LED * 3x Widerstand 320Ω * Kabel

Hintergrundinformationen

Steckbrett/ LED

Aufbau

LEDs auf den mittleren Teil des Steckbretts stecken (merken wo das kurze Bein ist!)
Welche Reihe auf dem Steckbrett gewählt wird ist egal.
Die Reihenfolge der LEDs entspricht den Farben der Ampel.

Reihen mit den kurzen Beinen mit Hilfe des Widerstands mit dem äußeren Rand verbinden.
Welche Reihe auf dem äußeren Rand genommen wird ist egal.

Äußere Reihe mit dem orangenen Kabel mit dem Raspberry Pi verbinden
(Am Raspberry Pi: äußere Reihe, 3. Pin von der Ecke aus gesehen)

LED mit dem jeweils gleichfarbigen Kabel mit dem Raspberry Pi verbinden.
Das Kabel muss in die Reihe mit dem langen Bein der LED gesteckt werden.

  • rot: Pin 4
  • gelb: Pin 5
  • grün: Pin 6

Bescheid geben um den Raspberry Pi zu starten.

Website: http://localhost:8080

Aufgabe:
Es wird eine Funktion implementiert, die als Parameter den LED Namen ('rot', 'gelb' oder 'gruen') und einen Status ('on' oder 'off') als Eingabe bekommt. Die Parameter werden durch die Website erzeugt. In der Funktion wird geprüft welche LED gerade geschaltet werden muss und dann entweder 'schalteRot(status)', 'schalteGelb(status)' oder 'schalteGruen(status)' aufgerufen.

Hinweise: Benötigt wird eine If-Bedingung. Jede Programmanwendung endet mit Semikolon:

Semantik if-Bedingung: 

                            WENN bedingung DANN programmanweisung 
                            ODER WENN andere bedingung DANN andere programmanweisung
Syntax: 

                          function schalteLED(ledName, status) //=> Die Funktion hat die Parameter ledName und status.
                            if ('A' === ledName) {             //=> hier wird der Wert 'A' mit dem Inhalt der 
                                                               //   Variable ledName verglichen
                              Programmanweisung;
                            } else if ('B' === ledName) {      //=> wenn die erste Bedingung nicht erfüllt war
                                                               //   wird Wert 'B' mit Inhalt der Variable verglichen.
                              andere Programmanweisung;
                            }
                          }
Hilfreiche Befehle: 

                        schalteGelb(status);       => Schaltet die gelbe LED in den angebenen Status ('on' oder 'off')

Lösung


                        function schalteLED(ledName, status) {
                          if ('rot' === ledName) {
                            schalteRot(status);
                          } else if ('gelb' === ledName) {
                            schalteGelb(status);
                          } else if ('gruen' === ledName) {
                            schalteGruen(status);
                          }
                        }

Aufgabe:
Wenn die Funktion 'blinkeLED' aufgerufen wird soll die gelbe LED so oft blinken, wie in dem Wert 'anzahl' auf der Webseite angegeben.

Hinweise:
Benötigt wird eine While-Schleife. Eine While-Schleife wird wiederholt ausgeführt so lange eine bestimmte Bedingung erfüllt ist.

Syntax: 

                            let zaehler = 0;            // Variable wird mit Wert 0 initialisiert
                            while(zaehler < anzahl) {   // Prüfung ob die Bedingung noch erfüllt ist
                                zaehler = zaehler + 1;  // Variable wird um den Wert 1 erhöht
                                Programmanweisung
                            }
Hilfreiche Befehle: 

                        await sleep(1);       => Wartet eine Sekunde bis die nächste Zeile ausgeführt wird.

Lösung


                        async function blinkeLED(anzahl) {
                          let zaehler = 0;
                          while(zaehler < anzahl) {
                            schalteGelb('on');
                            await sleep(1);
                            schalteGelb('off');
                            await sleep(1);
                            zaehler = zaehler + 1;
                          }
                        }

Datei:   server/workshop.js

Aufgabe:
Die Ampel soll in den Zustand 'rot' oder 'gruen' geschaltet werden. Die Ampel kann mit einem Mausklick auf die grüne oder rote Lampe auf der Website geschaltet werden.
Beim Schaltvorgang soll die Ampel drei Sekunden auf gelb stehen.

Hinweise:
Prüfen ob gerade die rote oder grüne LED angeschaltet ist. Je nachdem welche an ist kann die Ampel dann geschaltet werden, wenn sie nicht ohnehin schon im Zielzustand ist. (Bspw. wenn man zweimal auf grün drückt)

Hilfreiche Befehle: 

                                - getState('rot')           => liefert 'on' oder 'off' für die jeweilige LED zurück
                                - schalteRot('on')          => schaltet die angegebene LED in den Zustand
                                - await sleep(3)            => Wartet für drei Sekunden

Lösung


                      async function schalteAmpelZu(ledName) {
                        // Die LED soll zu dem Wert von 'ledName' geschaltet werden.
                        // Der Wert ist entweder 'rot' oder 'gruen'.
                        if ('rot' === ledName) {
                          if (getState('rot') === 'off') {
                            schalteLED('gelb', 'on');
                            schalteLED('gruen', 'off');
                            await sleep(3);
                            schalteLED('rot', 'on');
                            schalteLED('gelb', 'off');
                          }
                        } else if ('gruen' === ledName) {
                          if (getState('gruen') === 'off') {
                            schalteGelb('on');
                            await sleep(3);
                            schalteRot('off');
                            schalteGelb('off')
                            schalteGruen('on');
                          }
                        }
                      }
